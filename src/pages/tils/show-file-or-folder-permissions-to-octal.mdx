You want to show octal permission of a file or a folder.
Use `stat`

```shell
stat -c "%a" file
> 664
stat -c "%a %A %n" file
> 664 -rw-rw-r-- readme.md
```

This alias is handy.

```shell
alias octal="stat -c '%a %A %n'"
```
